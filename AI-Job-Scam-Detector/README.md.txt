# AI Job Scam Detector

AI powered system to detect fraudulent job offers using rule based analysis and domain verification.

## Features

- AI scam detection engine
- Email domain verification
- Screenshot OCR scanning
- PDF offer letter analysis
- Risk scoring system
- Admin analytics dashboard
- Scan history tracking

## Tech Stack

Frontend
- Angular
- Bootstrap

Backend
- Spring Boot
- REST APIs
- JWT Authentication

Database
- MySQL

AI Engine
- Custom Fraud Detection Engine

OCR
- Tesseract OCR

## Architecture

Angular UI  
↓  
Spring Boot REST API  
↓  
MySQL Database

### Backend

### Frontend
